**Money Manager** is a web application designed for personal finance management. It enables users to track budgets,
record transactions (income and expenses), organize categories, and generate reports.
The application focuses on ease of use, data security, and efficient handling of financial data.

Written in Java with Spring boot and Vaadin. Current version is available online. [Try it out!](https://moneymanager.sknt.ru)

### Key Features

- **Budgets**: Create, update, and delete budgets tied to specific categories or overall income/expenses. 
Supports renewable budgets with automatic date calculations for periods;
- **Transactions**: Track your income and expenses, associate them with categories. Provides summaries such as total amounts, balances,
and filtering by date ranges, types, and categories. Supports pagination for large datasets;
- **Categories**: Manage custom categories for transaction classification. View statistics including transaction counts
and amounts broken down by income/expenses;
- **Reports and Analytics**: Calculate overall summaries and analyze data by periods to identify spending patterns.
